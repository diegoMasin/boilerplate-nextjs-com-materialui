# Boilerplate Nextjs + MaterialUi
Estudando NextJS pela primeira vez, criei um boilerplate de NextJs com MaterialUI e StyledComponent. Login e Logout incluídos com Dashboard pós login.

#### Install Project

`yarn`

#### Run Project Local

`yarn dev`

#### Build Project

` yarn build`

#### Run Project Production

` yarn start`
